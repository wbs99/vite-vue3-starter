# AGENTS.md

基于 Vite 构建的 Vue 3 Web 应用 starter 模板，包含路由、状态管理、数据获取和 UI 组件。

## 快速开始

- 包管理器: pnpm (v9.6.0+)
- 开发: `pnpm dev` (端口 5555)
- 构建: `pnpm build`
- 预览: `pnpm preview`
- 代码检查: `pnpm lint`

## 版本控制

本项目使用 **jj (Jujutsu)** 进行本地版本控制（与 Git 并存，colocated 模式）。所有本地版本控制操作请使用 jj 命令，而非 git 命令。详见 `onevcat-jj` skill。

- **commit message 只写一行中文标题**，遵循 conventional commits 格式，如 `feat: 新增用户登录`、`fix: 修复空指针异常`
